## 🇧🇷 Português

### Sobre

Exercícios práticos desenvolvidos no componente **Linguagem de Programação I**, do primeiro período do curso superior de **Tecnologia em Análise e Desenvolvimento de Sistemas** no **Instituto Federal do Paraná (IFPR)**.

O objetivo não é demonstrar código de produção, mas sim registrar a evolução no aprendizado de lógica, tipos de dados, entrada/saída e estruturas de controle em C. Este repositório será atualizado conforme o curso avança.

### Exercícios

| Arquivo | Descrição |
|---|---|
| `imcCalc.c` | Calcula o IMC e classifica o resultado em faixas |
| `bhaskaraFormula.c` | Resolve equações de segundo grau pela fórmula de Bhaskara |
| `cpfVerifier.c` | Valida um CPF calculando os dois dígitos verificadores |
| `triangleIdentifier.c` | Identifica o tipo de triângulo (equilátero, isósceles ou escaleno) |
| `trapezeArea.c` | Calcula a área de um trapézio |
| `distanceConverter.c` | Converte distância em metros para km e estima custo de combustível |
| `hourConvert.c` | Converte horas totais em dias e horas restantes |
| `loginSystem.c` | Sistema simples de login com dois perfis (user e admin) |

### Contexto

- **Linguagem:** C (C99)
- **Fase:** Primeiro período — Linguagem de Programação I
- **Foco:** Lógica básica, tipos primitivos, `scanf`/`printf`, condicionais e laços

## 🇺🇸 English

### About

Practical exercises developed in the **Programming Language I** component, first semester of the **Technology in Systems Analysis and Development** program at **Instituto Federal do Paraná (IFPR)**, Brazil.

The goal is not production-ready code, but to document the learning progression in logic, data types, I/O, and control structures in C. This repository will be updated as the program advances.

### Exercises

| File | Description |
|---|---|
| `imcCalc.c` | Calculates BMI and classifies the result into health ranges |
| `bhaskaraFormula.c` | Solves quadratic equations using the quadratic formula |
| `cpfVerifier.c` | Validates a Brazilian CPF number by computing both check digits |
| `triangleIdentifier.c` | Identifies triangle type (equilateral, isosceles, or scalene) |
| `trapezeArea.c` | Calculates the area of a trapezoid |
| `distanceConverter.c` | Converts meters to km and estimates fuel cost for a trip |
| `hourConvert.c` | Converts total hours into days and remaining hours |
| `loginSystem.c` | Simple login system with two profiles (user and admin) |

### Context

- **Language:** C (C99)
- **Phase:** First semester — Programming Language I
- **Focus:** Basic logic, primitive types, `scanf`/`printf`, conditionals and loops
